# Stage Call — Thematic Upgrade Blueprint
Date: 2026-02-24  
Mode: Ideation only (no implementation changes)  
Primary Direction: Festival Poster Energy

## 1) The Crux (Brand Design, Distilled)
**Stage Call is the playable adrenaline of producing a live festival setlist under pressure.**  
Not “a cute routing game.” Not “a generic music skin.” It is **show control as choreography**.

If a new visual or feature does not increase this feeling, it is off-brand.

### Brand Test (Pass/Fail)
1. Does it feel like a festival poster came alive?
2. Does it improve split-second readability during chaos?
3. Does it heighten emotional momentum (build, drop, release)?
4. Does it preserve one-verb clarity: draw routes, make the show happen?

---

## 2) Creative North Star
Stage Call should feel like:
- **The map:** a printed festival plan marked up by an over-caffeinated stage manager.
- **The HUD:** production control strips and ticket stubs, not sterile app chrome.
- **The motion:** beats and crescendos, not random UI animation.
- **The audio:** crowd pressure + musical identity + tactical cues.

Core emotional loop:
1. **Authority:** “I’m running this event.”
2. **Flow:** “I can keep this chaos organized.”
3. **Euphoria:** “I just landed a perfect run.”

---

## 3) Brand System Architecture
To support many festivals without fragmenting identity, separate the system into **Invariants** and **Expressions**.

## 3.1 Invariant Scaffold (always true)
- Portrait-first composition and one-touch path gameplay.
- Instant cause/effect feedback on pathing, hazards, delivery, and combo.
- Stage role clarity (large/medium/small) and scoring logic readability.
- “Production energy” UI language (set, slot, crowd pressure, run total).
- High-contrast gameplay layer over visually rich but recessive map art.
- Same interaction grammar across festivals (menu, round, fail, complete).

## 3.2 Festival Expression Layer (changes per festival)
- Palette family and accent behavior.
- Typography accent font and icon motif.
- Stage/distraction visual language.
- Background map storytelling and geometry flavor.
- Audio palette (instrumentation, crowd density, timbral color).
- Flavor copy and naming.

Design principle: **different posters, same promoter.**

---

## 4) Visual Identity Overhaul
## 4.1 Typography Strategy
- **Display voice:** tall condensed, assertive, uppercase for event moments.
- **System voice:** clean geometric sans for tactical numbers and controls.
- **Festival accent voice:** optional third type treatment for map-specific flavor (used sparingly in menus/cards only).

Use typography as hierarchy, not decoration:
- Title = headliner billing.
- Score + encounters + timer = production telemetry.
- Combo callouts = crowd hype inserts.

## 4.2 Color Grammar
Global semantic tokens:
- `success` (delivery, streak confidence)
- `warning` (timer risk, hazard escalation)
- `danger` (critical pressure)
- `neutral-map` (recessive terrain)
- `energy-accent` (festival-specific signature)

Festival packs remap accents but never semantic meaning.  
Result: players relearn style quickly but never relearn state logic.

## 4.3 Shape + Texture Language
- Rounded ticket chips, poster edge tears, diagonal separators.
- Inked route lines with slight analog texture.
- Subtle print grain + layered paper depth on overlays.
- Avoid glossy mobile skeuomorphism and over-neon blur spam.

---

## 5) Motion + Interaction Feel
Motion should follow a music metaphor:
- **Intro (menu/start):** staged reveal, low BPM confidence.
- **Round rhythm:** snappy tactical pulses tied to player actions.
- **Combo escalation:** amplitude and density increase.
- **Fail/win punctuation:** one strong hero beat, then settle.

Rules:
- Every animation must either inform state or amplify emotion.
- Keep pathing and collision readability above all flourish.
- Prioritize “kinetic certainty” over floaty easing.

---

## 6) Thematic UI Framework (No Build, Direction Only)
## 6.1 In-Round HUD
Reframe as a compact show-control strip:
- `Set Score`
- `Crowd Pressure` (encounters left / strike pressure)
- `Time to Slot`
- `Level + Run context`

Design behavior:
- Calm at baseline.
- Gains visual urgency with meaningful events only.
- Never blocks path planning zones.

## 6.2 Feedback System
- Delivery: punchy burst + crowd-lift cue + numeric pop.
- Collision/distraction: social friction bubble language (thought/speech styled).
- Combo: stage-local hype ladder with clear multiplier legibility.
- Miss: disappointment signal without tonal collapse.

## 6.3 Menu and Festival Select
Treat each festival card like a lineup poster:
- Strong identity art.
- “Tonight’s profile” summary.
- Progress and best score framed as “tour stats.”

## 7) Multi-Festival Theme System
The game scaffold stays fixed. Festival identity varies via curated modules.

## 7.1 Theme Modules Per Festival
1. Palette pack  
2. Type accent pack  
3. Stage architecture set  
4. Distraction cast set  
5. Map motif layer  
6. Audio identity pack  
7. Copy flavor pack

## 7.2 Proposed Festival Archetypes
### A) Urban Pulse (Gov Ball DNA)
- Mood: street poster + park concrete/grass contrast.
- Palette: electric orange, cyan, lime on muted terrain.
- Geometry: crossing boulevard paths, dense central conflict.
- Audio: percussive electronic + crowd chatter.

### B) Desert Mirage
- Mood: dusty sun-bleached spectacle.
- Palette: sand, coral, turquoise, sunset magenta.
- Geometry: sweeping routes, wide arcs, exposed risk lanes.
- Audio: warm percussion, airy vocals, analog synth shimmer.

### C) Neon Dockyard
- Mood: industrial night rave.
- Palette: steel blue, hazard yellow, acid green, red alarm accents.
- Geometry: channels, bridges, bottlenecks, hard corners.
- Audio: punchy bass, metallic transients, tight rhythmic loops.

### D) Woodland Ritual
- Mood: mystical daytime-to-dusk gathering.
- Palette: moss, clay, lantern amber, moonlit teal.
- Geometry: organic branching, circular clearings, soft choke points.
- Audio: hand percussion, organic textures, vocal swells.

### E) Midnight Warehouse
- Mood: underground maximalism.
- Palette: charcoal, ultraviolet accents, sodium orange emergency tones.
- Geometry: compressed pathways, sudden congestion zones.
- Audio: high-energy club pulse, aggressive crowd surges.

## 8) Map Art Direction System
Each map should be designed as **playable composition**, not background art.

Required map roles:
1. **Hero stage anchor:** immediate visual focal point.
2. **Chaos corridor:** where emergent conflicts are likely.
3. **Recovery lane:** safer reroute option with longer travel.
4. **Risk-reward pocket:** high score potential with hazard overlap.

Map storytelling should signal movement intention:
- Routes imply flow direction.
- Decorative props support fantasy but do not obscure actors.
- Distraction zones read instantly as temptation/risk.

---

## 9) Audio Art Direction
Global identity:
- Music arc follows level intensity bands (early/mid/peak).
- SFX blend musicality and legibility; never purely realistic.
- Crowd is a “living pressure bed,” not constant noise wall.

Festival-specific identity:
- Distinct instrument families per festival.
- Shared cue semantics (delivery, miss, combo, hazard) for consistency.
- Dynamic mix behavior tied to run state and combo escalation.

Audio principle:
**Tension should sound exciting, not exhausting.**

---

## 10) Copy + Voice Direction
Brand voice: confident, event-forward, energetic, concise.

Examples:
- “Start Festival” becomes contextually richer where appropriate:
  - “Open Gates”
  - “Run the Set”
  - “Backstage Ready”
- Keep tactical labels clear; use flavor in wrappers and transitions.

Copy guardrails:
- No snarky meme tone.
- No bland utilitarian enterprise tone.
- Sound like a world-class live events crew.

---

## 11) Admin and Asset Governance (Creative Ops)
Given sprite/audio prompt workflows, define creative control standards:

1. **Prompt registry** per festival: canonical prompts, variants, rejects.
2. **Visual acceptance checklist:** silhouette clarity, scale read, palette compliance, style coherence.
3. **Audio acceptance checklist:** recognizability, fatigue profile, mix compatibility.
4. **Swap policy:** any override must be reviewed against brand test + gameplay readability.
5. **Versioned style bible:** each festival has a one-page design token and motif sheet.

This keeps experimentation high while protecting brand consistency.

---

## 12) What “Great” Looks Like
A top-tier Stage Call build should make a player say:
- “This feels like I’m directing a real festival.”
- “I can read everything instantly even when it gets wild.”
- “Every festival feels fresh, but I still feel at home.”

That is the target intersection:
**identity, readability, and emotional velocity.**
